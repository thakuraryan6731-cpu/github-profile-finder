# GitHub Profile Finder

A responsive web application that fetches GitHub user data using the GitHub REST API.

## Features

- Search GitHub users
- Display avatar, bio, followers, following and repositories
- Open GitHub profile in a new tab
- Loading indicator
- Error handling
- Press Enter to search
- Responsive UI

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- GitHub REST API
