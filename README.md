# 🚀 GitHub Profile Finder

A modern web application that allows users to search for any GitHub profile using the GitHub REST API. It displays profile information such as avatar, bio, followers, repositories, and provides a direct link to the user's GitHub profile.

## ✨ Features

- 🔍 Search GitHub users by username
- 👤 Display profile avatar
- 📛 Show full name
- 📝 Display bio
- 👥 Followers & Following count
- 📦 Public repositories count
- 🔗 Open GitHub profile in a new tab
- ⌨️ Press **Enter** to search
- ⏳ Loading indicator while fetching data
- ❌ Error handling for invalid usernames
- 📱 Responsive design
